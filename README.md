## 介绍
这是一款能够自定义签到方式的校园考勤位置签到的APP,只因作者实在受不了学校官方弄的签到公众号,又卡又不稳定,有时候还签不上,于是另起炉灶自己按照抓包来的RESTFul格式编写了这款好看又强大的签到APP

## 使用
非常简单
- 初次打开APP时先进入设置登录学号和密码
- 然后下一次打开APP就点"开始签到"按钮就行了
